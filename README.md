# FinPulse Backend

Микросервисная бэкенд-платформа для финансового анализа и уведомлений.

## Архитектура

Проект представляет собой многомодульное Maven-приложение, состоящее из следующих микросервисов:

### User Service (`userservice`)
- Регистрация и аутентификация пользователей
- Управление профилями пользователей
- Управление сессиями и доступом

### Aggregation Service (`aggregationservice`)
- Взаимодействие с внешними API (ГОСТ-шлюз / Open Banking API)
- Агрегация финансовых транзакций
- Публикация заданий в Kafka (топик `user_forecast_update`) для Python ML-сервиса

### Notification Service (`notificationservice`)
- Управление уведомлениями пользователей
- Интеграция с Kafka для обработки событий

## Технологический стек

- **Java 17** + **Spring Boot 3.4.4** + **Spring Cloud 2024.0.1**
- **PostgreSQL** - хранение пользовательских данных и транзакций
- **Redis** - кэширование и управление сессиями
- **MinIO** - хранение отчетов и экспортов
- **Apache Kafka** - асинхронная коммуникация между сервисами
